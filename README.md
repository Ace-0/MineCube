# MineCube
 A sample voxel editor based on OpenGL 3.3+, inspired by [MagicaVoxel](https://ephtracy.github.io/).



## Dependences

1. OpenGL 3.3+
2. GLAD
3. GLFW
4. [imgui](https://github.com/ocornut/imgui)
